# Сборка в программу для Windows

## Что нужно установить

1. **Node.js 20+** — https://nodejs.org (выбрать LTS)
2. **Git** — https://git-scm.com (если нет)

---

## Пошаговая инструкция

### Шаг 1 — Скачайте проект

Нажмите **Скачать → Скачать код** на платформе poehali.dev.  
Распакуйте архив в удобную папку, например `C:\muzvocal`.

### Шаг 2 — Откройте терминал в папке проекта

Правой кнопкой по папке → **"Открыть в терминале"** (или PowerShell).

### Шаг 3 — Установите зависимости

```bash
npm install
```

### Шаг 4 — Соберите Windows-программу (одна команда)

```bash
npx vite build --config vite.electron.config.ts && npx electron-builder --win
```


## Частые вопросы

**Q: При сборке ошибка про Python?**  
A: Установите Python 3.x с https://python.org — он нужен для компиляции некоторых модулей.

**Q: Антивирус блокирует .exe?**  
A: Это нормально для неподписанных приложений. Добавьте в исключения или используйте портативную версию.

**Q: Можно ли использовать микрофон в программе?**  
A: Да. При первом запуске Windows запросит разрешение на доступ к микрофону — разрешите.

**Q: Треки сохраняются локально или в облаке?**  
A: Треки хранятся в облаке (S3). Для работы нужен интернет.

## Структура Electron-файлов

```
electron/
  main.js       — главный процесс (окно приложения), ES module
  preload.cjs   — мост безопасности между Electron и веб, CommonJS
electron-builder.json      — настройки сборки
vite.electron.config.ts    — конфиг Vite с base: './' для Electron
```
